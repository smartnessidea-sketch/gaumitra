# GauMitra Website

Static website for `gaumitra.world`.

## Files
- `index.html` - website content
- `style.css` - responsive premium design
- `script.js` - 23 clickable feature modules with modals
- `assets/logo.jpeg` - GauMitra logo
- `CNAME` - GitHub Pages custom domain file

## Publish
Upload all files to a GitHub repository, enable GitHub Pages from the main branch, and set custom domain to `gaumitra.world`.
